# 📦 SIMBARANG - Inventory Management System

Web-based inventory management system developed to improve efficiency and accuracy in managing warehouse data at UPT PPD Kabupaten Kapuas.

---

## 📌 Project Overview
The inventory management process at UPT PPD Kabupaten Kapuas was previously done manually using books and spreadsheets, which caused issues such as data inaccuracies, delayed reporting, and difficulty tracking assets.

This project aims to develop a web-based system (SIMBARANG) to digitize and streamline inventory management processes.
